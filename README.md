# Customer Shopping Behaviour Analysis

An end-to-end data analytics project using Python for data cleaning, MySQL for database querying, and Power BI for creating an interactive dashboard to analyze retail customer trends.

---

## Project Objectives
* **Customer Segmentation:** Grouping customers into loyalty segments to see who shops the most.
* **Discount Impact:** Checking if customers using discounts are spending more than the average shopper.
* **Product Performance:** Tracking the highest-rated and most-purchased items across different store categories.
* **Shipping Trends:** Comparing how shipping methods (Express vs. Standard) relate to the total purchase amount.

---

## Tools Used
* **Python (Pandas, NumPy):** Data cleaning and feature engineering.
* **SQLAlchemy:** Programmatically exporting data from Python into MySQL.
* **MySQL:** Advanced data querying and aggregations.
* **Power BI:** Data visualization and KPI dashboard building.

---

## Data Cleaning & Staging (Python)
Before analyzing the data, the raw dataset was processed in the `Customer Shopping Analysis.ipynb` notebook:
* **Handling Missing Values:** Filled missing review ratings using the median rating of that specific product category.
* **Removing Duplicates:** Dropped the `promo_code_used` column because it was an exact duplicate of the `discount_applied` column.
* **Feature Engineering:** * Created an `age_group` column to split customers into Young Adults, Adults, Middle-aged, and Seniors.
  * Mapped shopping frequencies (Weekly, Monthly, etc.) into a numeric `purchase_frequency_days` column.
* **Database Export:** programmatically sent the cleaned data directly into MySQL using SQLAlchemy.

---

## Data Analysis & Insights (SQL)
The queries inside `Customer_behaviour_query.sql` extract key business metrics:
* **Loyalty Tiers:** Grouped customers based on previous purchases (New = 1, Returning = 2-10, Loyal = 10+).
* **Top Products:** Used window functions (`ROW_NUMBER`) to find the top 3 most-purchased items inside each product category.
* **Revenue Breakdown:** Calculated total revenue generated by gender and different age groups.
* **Promo Tracking:** Filtered for customers who used a discount but still spent more than the store's average order value.

## Repository Files
* `customer_shopping_behavior.csv`: The raw dataset used for the project.
* `Customer Shopping Analysis.ipynb`: Jupyter notebook covering the data cleaning and database export.
* `Customer_behaviour_query.sql`: SQL script containing all the analytical database queries.
* `Power BI Dashboard.pbix`: The final interactive Power BI dashboard file.

---

## How to Run This Project
1. Open and run `Customer Shopping Analysis.ipynb` to clean the data and load it into your local MySQL server.
2. Execute the queries in `Customer_behaviour_query.sql` to view the aggregated data metrics.
3. Open `Power BI Dashboard.pbix` in Power BI Desktop to explore the visual charts.
