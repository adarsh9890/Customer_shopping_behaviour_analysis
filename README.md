# Customer_behaviour_analysis
Data analytics project showcasing customer behaviour analysis using python, sql and power bi.
Overview
This project covers the complete data analytics workflow: loading a dataset in Python, performing EDA, cleaning data, running SQL queries, and building a Power BI dashboard to present key insights.

Dataset
Source: Provided dataset (CSV/Excel)
Contains customer, sales, and product-related details
Used for analysis, cleaning, and visualization

Tools Used
Python,SQL,Power BI,Jupyter Notebook

Steps
1.Loaded dataset in Python
2.Performed EDA and cleaned the data
3.Wrote SQL queries for deeper analysis
4.Built an interactive Power BI dashboard to visualize insights

Dashboard
Interactive Power BI dashboard showing key metrics like sales trends, customer behavior, and category performance.

Results
Clean and structured dataset
Insights into trends, patterns, and customer behavior
Easy-to-understand visualizations using Power BI

How to Run
Open the .ipynb notebook in Jupyter
Install required libraries from requirements.txt
Run SQL queries in your database client
Open the .pbix file in Power BI Desktop to view the dashboard
